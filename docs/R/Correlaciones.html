Condiciones para la aplicación la prueba de Pearson

  1. La muestra debe ser elegida de manera aleatoria de la población que representa.
  2. Las variables deben ser cuantitativas (de intervalo o razón)
  3. Las variables deben tener una distribución normal bivariada
  4. Los residuos (error) deben presentar homocedasticidad (ausencia de heterocedasticidad)

Prueba de normalidad bivariada

El valor de p es inferior a 0.05, por tanto, la distribución NO es normal

Coeficiente \(\rho\) de Spearman y coeficiente \(\tau\) de Kendall

Se usan cuando los datos no son lineales pero son monotónicos

Los datos

lex <- c(47, 89, 131, 186, 245, 284, 362, 444, 553, 627)

gram <- c(0, 2, 1, 3, 5, 9, 7, 16, 25, 34)

La hipótesis

H\(_{1}\):Hay una correlación positiva entre el tamaño del vocabulario y la complejidad gramatical.

H\(_{0}\):No hay correlación entre el tamaño del vocabulario y la complejidad gramatical.

plot(gram ~ lex, main = "Tamaño del vocabulario y complejidad gramatical", xlab = "Tamaño del vocabulario", ylab = "Complejidad gramatical")

lines(lowess(gram ~ lex))

Calculamos el coeficiente

cor(gram, lex, method = "spearman")
[1] 0.9757576
cor(gram, lex, method = "kendall")
[1] 0.9111111

Significatividad del coeficiente

Condiciones del test de significatividad

  1. La muestra ha sido extraída aleatoriamente de la población que representa.
  2. Las variables están en escala ordinal
